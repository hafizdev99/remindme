How to Install Nasıl Kurulur

Takes less than 60 seconds — no developer skills required 60 saniyeden az sürer — geliştirici bilgisi gerekmez

1
Download the Extension
Eklentiyi İndirin
Click the Download button below and save the remindme-extension.zip file anywhere on your computer.
Aşağıdaki İndir düğmesine tıklayın ve remindme-extension.zip dosyasını bilgisayarınızdaki herhangi bir yere kaydedin.
2
Extract the ZIP File
ZIP Dosyasını Çıkartın
Right-click the downloaded ZIP file and select "Extract All" (Windows) or double-click (Mac). You'll get a folder called reminder-extension.
İndirilen ZIP dosyasına sağ tıklayın ve "Tümünü Çıkart"ı seçin (Windows) veya çift tıklayın (Mac). reminder-extension adında bir klasör elde edersiniz.
3
Open Chrome Extensions
Chrome Uzantılarını Açın
In Chrome, go to chrome://extensions — paste this in the address bar and press Enter. Or go to Menu → Extensions → Manage Extensions.
Chrome'da chrome://extensions adresine gidin — bunu adres çubuğuna yapıştırın ve Enter'a basın. Ya da Menü → Uzantılar → Uzantıları Yönet yolunu izleyin.
4
Enable Developer Mode
Geliştirici Modunu Etkinleştirin
In the top-right corner of the Extensions page, toggle on Developer mode. New buttons will appear in the top-left.
Uzantılar sayfasının sağ üst köşesinde Geliştirici modunu açın. Sol üstte yeni düğmeler belirecektir.
5
Load the Extension
Eklentiyi Yükleyin
Click Load unpacked and select the extracted reminder-extension folder. RemindMe will appear in your extensions list instantly!
Paketlenmemişi yükle'ye tıklayın ve çıkartılan reminder-extension klasörünü seçin. RemindMe uzantı listenizde anında görünecek!
6
Pin It to Toolbar
Araç Çubuğuna Sabitleyin
Click the puzzle-piece icon (🧩) in Chrome's toolbar → find RemindMe → click the pin icon. The ⏰ icon will appear in your toolbar — you're done!
Chrome'un araç çubuğundaki yapboz simgesine (🧩) tıklayın → RemindMe'yi bulun → raptiye simgesine tıklayın. ⏰ simgesi araç çubuğunuzda görünecek — hazırsınız!
